TvExplorer privacy policy

TvExplorer does not collect users’ private information, such as Google/Amazon usernames, accounts, etc, except for functionality provided by the application.
TVExplorer does not show ads.
TvExplorer does not access or use device location.

TvExplorer can:

●     read/write/delete in the device's internal or external memory

●     send or receive data over the Internet (as a result of user interaction with the app)

●     play music or video

●     and provide all the functions as described on the TvExplorer website

In return for some of the premium features of TvExplorer, you may choose to be a peer on the Bright Data network.

By doing so you agree to have read and accepted the Terms of Service of the Bright SDK EULA:https://bright-sdk.com/eula and Bright Data’s Privacy Policy: https://brightdata.com/legal/sdk-privacy.

You may opt out of the Bright Data network from the settings menu.

Explanation of needed permissions

TvExplorer requests the following permissions for a given purpose:

●     Internet - for reporting crashes

●     Read/write external storage - for file management

●     Record Audio - for voice search

Crash reports

When TvExplorer crashes, it sends the crash report, which is a precious tool for the developer of TvExplorer to fix problems and make the app better.

Crash reports are stored on the developer's server for the minimum needed time (about 1 month) and are used solely for fixing app problems.

The crash report contains the following data:

●     date of posting

●     app version, android version

●     device model (name)

●     stack trace - actual information about the crash, which is typically up to 15 lines of locations into the app's source code where the crash happened; this information doesn't store anything personal
