Key Features

Auto-Scan USB Drives

Automatically detects and lists all media/video files from your USB drive with thumbnail previews, file names, progress, duration, and file size.

Smart Playback Resume

Pick up right where you left off! Get prompted to resume or start over for previously watched videos.

Subtitle Support

Select from multiple subtitle formats including SRT, VTT, ASS, SSA, TTML, DFXP, and XML for enhanced viewing.

Advanced Player Controls

Clean player interface with rewind/forward (5 seconds), subtitle selection, progress bar with timestamps, and playback controls.

Wide Format Support

Supports MP4, AVI, MKV, MOV, FLV, WMV, WebM, M4V, 3GP, TS, MTS, M2TS, VOB, and OGV formats.

Reload Files

Quickly reload and rescan your USB drive to detect newly added files without restarting the app.
