Frequently Asked Questions

How do I transfer PDF files to my Android TV? +

Photo Dream Screensaver offers multiple easy transfer methods: 1) Web import with secure token - scan QR code and upload from any device without network restrictions, 2) WiFi upload for wireless file transfer, 3) FTP import when devices are on the same network. No USB drive required!

What is the web token import method? +

Web token import is a secure, convenient way to upload PDFs from any device. Simply scan the QR code displayed on your TV, enter the token on your phone or computer, and upload files - no shared network needed! This is the most flexible method and works from anywhere.

Can I use WiFi to transfer files? +

Yes! The WiFi file upload feature allows you to wirelessly transfer PDFs from your phone, tablet, or computer to your Android TV. Just scan the QR code, connect via your browser, and upload files instantly over your WiFi network.

Is Photo Dream Screensaver optimized for TV remote controls? +

Absolutely. Photo Dream Screensaver is designed specifically for Android TV remote navigation. Use left/right keys to navigate pages, long-press OK for file options, and access all features through an intuitive overlay menu. Everything is optimized for 10-foot UI experience.

Can I present PDFs in classrooms or meetings? +

Yes! Photo Dream Screensaver is perfect for presentations, classrooms, and meeting rooms. Display PDF presentations, lecture notes, or documents on any Android TV for easy viewing by large audiences. The night mode and zoom features make it ideal for professional use.

Does the app support FTP transfers? +

Yes! Photo Dream Screensaver supports FTP import with anonymous connection. This is great for professional environments where you need to transfer multiple files or integrate with existing workflows. Your TV and computer need to be on the same network for FTP.

Does the app support night mode? +

Yes! Photo Dream Screensaver includes a comfortable night mode feature for reduced eye strain during low-light viewing conditions. Perfect for reading e-books or documents in the evening.
