Frequently Asked Questions

What is TV PPTX Viewer? +

TV PPTX Viewer is an Android TV app for opening and viewing PPTX PPTX files on a TV using a remote-friendly interface.

How can I add PPTX files to the app? +

You can import PPTX files using Web Import, WiFi Upload, FTP Import, or by scanning available PPTX files from connected storage such as USB.

Does Web Import require the phone or laptop to be on the same WiFi network as the TV? +

No. Web Import uses a token-based flow, so the sending device does not need to be on the same network as the TV.

When should I use WiFi Upload instead of Web Import? +

Use WiFi Upload when your sending device and Android TV are on the same network. It is designed for faster local file transfer.

Does FTP Import need the same network? +

Yes. FTP Import uses an anonymous FTP connection and requires the sending device to be on the same network as the TV.

Can I bookmark slides? +

Yes. You can bookmark the current slide, view bookmarks for the current PPTX, and also browse bookmarks across all PPTX files.

Does the app remember where I stopped? +

Yes. The viewer can reopen a PPTX with the last viewed slide number and last used view settings such as zoom and night mode.

Can I search inside PPTX files? +

Yes. The viewer includes a search option to find text inside supported PPTX files.

Can I delete imported PPTX files? +

Yes. Long-press OK on a focused PPTX file to view file info and options such as open and delete. Settings also include an option to clear all PPTXs in the imports folder. USB files are not deleted.

Where can I manage subscription information? +

The app includes a subscribe page and a manage subscription page with information about managing your subscription.
